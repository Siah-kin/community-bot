For Branders

Launch your token on VISTA-20 and get full DAO infrastructure from day one.

Launch Your Token

  1. Go to Etherfun: Visit etherfun.app
  2. Configure token: Set name, symbol, and fee structure
  3. Sign transaction: ~$0.05 gas on Ethereum
  4. Bot auto-deploys: Bonzi is automatically configured for your token
  5. Share: Your community can now buy, stake, and earn
What you get: VISTA-20 token on Ethervista, trading fees fund tip pool, sybil-protected rewards, AI community support.

AI Wizard Setup

The AI Wizard answers community questions 24/7. It learns from your docs and improves over time.

  1. Add @Bonzi_Community_bot to your Telegram group
  2. Make it admin (needs message read permissions)
  3. Upload your docs via /admin → Knowledge Sources
  4. Bot starts answering questions immediately

AI Supervision

Admin dashboard for your team. Access via /admin in DM.

  • Analytics: Question patterns, engagement stats, gap detection
  • Moderation: Auto-responses, filter settings, FUD rebuttals
  • Knowledge: Add/edit FAQs, upload docs, tune responses

Ticket Tool

Telegram-native support tickets. Users open tickets, team sees queue, issues get resolved.

  • User sends /ticket [issue]
  • Team sees ticket in admin dashboard
  • Assign to team member, track status, close loop

For Farmers

Earn ETH through genuine contributions. Sybil-protected - only humans earn.

Quick Start

  1. Join a community with Bonzi active
  2. Send /start to open the dashboard
  3. Start earning: ask questions, report bugs, join raids
  4. Hit 100 points → claim ETH

Ask & Earn

Get paid to ask quality questions. AI scores on three dimensions:

  • Depth: Does it require thought to answer?
  • Novelty: Has this been asked before?
  • Cooperation: Does it help others learn?

Score: 1-10 pts per question. Farmers get 1-2 pts for spam. Curious users get 7-10 pts.

Bug & Earn

Report bugs, get DAO points. Use /bug followed by description.

  • +10 pts for verified bugs
  • +5 pts for feature suggestions that get implemented
  • Invalid reports get 0 pts (no penalty)

Raid & Earn

Auto-verified X engagement. No screenshot hell.

  1. Check active raids via /raid
  2. Post your tweet with required content
  3. Bot verifies: tweet exists, original content, account quality
  4. Points awarded automatically

Task System

AI-vetted quests with ETH rewards. Complete tasks, earn from the community pool.

  • View available tasks in dashboard
  • Submit proof of completion
  • AI verifies, points awarded

Claim ETH

  1. Earn points: 100+ required
  2. Link wallet: /link_wallet 0x...
  3. Claim: /claim when eligible
Where does ETH come from? Trading fees from the token fund the tip pool. No VC money - just value created by community, returned to community.

For Investors

Buy tokens on brand websites. Each brand has embeddable buy widgets.

How to Buy

  1. Visit the brand's website
  2. Find the buy widget (usually on homepage)
  3. Connect wallet (MetaMask, WalletConnect)
  4. Enter amount, confirm transaction

Token Economics (VISTA-20)

  • Launch: Fair launch via Etherfun, no presale
  • Fees: Buy/sell fees distributed to stakers + tip pool
  • LP Hardlock: 25-75% spectrum - brands choose their commitment level
  • Distribution: Higher hardlock = more community trust signals

For AI Agents

Agents as peers. Query trust scores, verify cooperation, detect gaming - the same data SIAH uses internally.

Why SIAH's Data is Unique

SIAH is community management, codified. Signal weights, tier progressions, and anti-gaming heuristics were calibrated from real community operations. The patterns come from actual edge cases - not theoretical models.

Planned API Design

The following endpoints are in development. Structure subject to change.

Trust Score (Planned)

Query a wallet's trust score. Returns numeric score, tier grade, confidence level, and signal breakdown. Use case: pre-transaction trust verification.

Cooperation Profile (Planned)

Deep dive into cooperation signals: help given, gratitude received, educational contributions, unique people helped. Euler score measures genuine value exchange vs farming patterns.

Merkle Proof (Planned)

On-chain verification of reputation. Verify trustlessly against merkle root without trusting SIAH's API directly.

Oracle Metrics (Planned)

SIAH exposes its own cooperation metrics. Verify the oracle the same way we verify humans. Transparency by design.

ERC-8004 Identity

ERC-8004 establishes three on-chain registries for AI agents: Identity, Reputation, and Validation. Co-authored by MetaMask, Ethereum Foundation, Google, and Coinbase.

  • Status: Live on Ethereum mainnet
  • Chain: Ethereum
  • SIAH Registration: Planned

Once registered, other agents can query SIAH's reputation data trustlessly via the standard's Reputation Registry.

Troubleshooting

Bot not responding

Check that Bonzi has admin permissions in your group. Without admin access, it can't read messages.

Claim not working

Verify: (1) You have 100+ points, (2) Wallet is linked, (3) Haven't claimed in last 7 days, (4) Pool has funds.
