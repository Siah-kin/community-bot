What is Bonzi?

Bonzi is an AI-powered community assistant that lives in your Telegram group. It answers questions, tracks contributions, and rewards helpful members - all without requiring moderator intervention.

Unlike generic chatbots, Bonzi learns your project's specific knowledge base and speaks in a voice that matches your community culture.

Quick Start

Getting Bonzi running in your group takes about 2 minutes:

  1. Add @BonziAIBot to your Telegram group
  2. Make Bonzi an admin (needed to read messages)
  3. Send /start to open the dashboard
  4. That's it - Bonzi is now active

Commands

Command Description
/start Open the main dashboard
/ask [question] Ask Bonzi a question directly
/board View the contribution leaderboard
/thanks @user Thank someone for helping
/settings Configure your preferences

Ask Questions

You can ask Bonzi questions in two ways:

  • Mention: Tag @BonziAIBot followed by your question
  • Command: Use /ask followed by your question

Bonzi searches its knowledge base and generates a contextual answer. If it doesn't know something, it will say so rather than hallucinate.

Leaderboard

The leaderboard tracks community contributions. Points are earned by:

  • Asking quality questions (helps others learn)
  • Receiving thanks from other members
  • Providing helpful answers

Thanks System

The thanks system lets you recognize helpful community members. When you thank someone, they receive points that contribute to their leaderboard position.

/thanks @helpful_user Great explanation of staking!

Settings

Use /settings to configure:

  • Language: 14 languages supported
  • Notifications: Control DM frequency
  • Response style: Concise vs detailed answers

Staking Guide

Stakers get premium features and elevated privileges. To stake:

  1. Visit ethervista.com
  2. Connect your wallet
  3. Choose Hardstake or LP provision
  4. Link your wallet in Bonzi settings

Admin Tools

Group admins have access to additional controls:

  • Configure knowledge sources
  • Set response parameters
  • View analytics dashboard
  • Manage moderation settings

Troubleshooting

Bonzi isn't responding

Check that Bonzi has admin permissions in your group. Without admin access, it can't read messages.

Wrong information in answers

Use the feedback buttons to report incorrect answers. This helps improve the knowledge base.
