How the loop works

A public-safe walkthrough of the Bonzi journey: enter, contribute, get routed, and see what evidence comes back before deeper permissions exist.

Entry

The visible path starts with an application or signed bot link, then unlocks this static journey for the current browser session.

Evidence

The demo describes the proof shape without exposing founder-only thresholds, prompts, provider internals, or exploit tests.

Routing

Contributors see how a report or contribution moves from intake into review without gaining admin authority.

This page is informational. Access here does not grant payments, voting, operational control, or privileged backend data.
