Economic Glossary

Closed-Loop
Value returns to generators. No extraction layer. Forest nutrient cycle.
Hive
Hierarchical. Value flows up to queen/orchestrator. Workers expendable.
Swarm
Coordination pattern. Efficient but extractive by design.
Agent-Centric
Local consensus. Each agent validates locally, no global coordinator.
Data-Centric
Central authority aggregates and controls. Single point of failure.
Mutual Credit
Reciprocal obligations. Your credit is my debit. Net zero creation.
Self-Compounding
Helping others returns value to you. Generosity is economically rational.
Yield vs Fee
Yield = returns from real activity. Fee = extraction layer, value leaves system.
Plutocracy
Governance by capital. More tokens = more votes. Whale dominance.
Meritocracy
Governance by contribution. Reputation-weighted. Earned influence.
Actors vs Inter-actors
Actors observe. Inter-actors participate and have stake. Conversion via tip/task loop.
Antifragile
Gets stronger from stress. No single point of failure. Closed-loop property.
