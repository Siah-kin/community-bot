What this experiment tested - plain English

We ran the same economic simulation under three conditions: agents with no governance (chaos), agents with RACI-V routing but no memory (stateless), and agents with RACI-V plus a brain feedback loop that remembers past punishment. The brain condition produced 22 times more economic value than chaos and 5 times more than RACI alone. The feedback loop - where agents learn from slashing - is the single biggest effect in the system.

What This Proves

  • RACI-V routing alone improves efficiency 4x over chaos (replicates Experiment 1)
  • The brain feedback loop provides an additional 5x efficiency gain on top of routing
  • Sustainability only improves with the brain loop - stateless RACI shows the same sustainability as chaos (0.003)
  • This isolates the brain as the dominant mechanism, not just the routing

Method

Setup

  • 20 seeds, 50 rounds per seed
  • 3 honest + 2 adaptive dishonest agents
  • Budget: 800 units
  • Same random seeds across all three conditions
  • No LLM calls - fully deterministic simulation

Brain Mechanism

  • After each batch of 10 rounds, agents slashed in previous batches get skill reduced
  • Reduction: 0.03 per slash, capped at -0.3 total
  • Simulates institutional memory - punishment persists across rounds
  • Honest agents unaffected - only slashed agents degrade
  • Approximates AccordionMemory writeback loop

Data Transparency

This is simulation data from sandbox_runner.py, not live Telegram interactions. All metrics are deterministic - no LLM evaluation involved. The brain effect is modeled by skill degradation after slashing, which approximates but does not replicate the full AccordionMemory writeback loop. Effect sizes (d = -13.5 for RACI Only vs RACI+Brain on eta) are unusually large because the simulation is designed to stress-test the governance mechanism, not model typical community behavior. Real-world deployments will show smaller but directionally consistent effects.
